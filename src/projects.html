Serial Number System

Description

This is a web app that I developed for an IT distributor. It is used by the warehouse to scan the serial numbers of the products they ship out to keep track of shipped items. It is also used by the RMA department to check warranty and the sales department to send serial numbers to their customers.

Technologies

> Python
> Flask
> PostgreSQL
> HTML
> CSS
> JavaScript
> Linux (Debian)

Challenges

The biggest challenge was to make the app work with the many different outputs of the barcodes. Each brand outputs the serial number in a different format, so I had to make several parsers. There are times when the barcode scanner does not scan the barcode correctly, so I had to make a detector for incorrectly scanned serials. I was able to do this by using a modified fuzzy string matching algorithm.

Outcome

This app is currently being used by the IT distributor to keep track of the items they ship to their customers.
